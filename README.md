# elm-todomvc

An implementation of the [TodoMVC](http://todomvc.com/) application in
[Elm](http://elm-lang.org/).

# Usage

```sh
$ git clone https://github.com/dwayne/elm-todomvc.git
$ cd elm-todomvc
$ nvm use
$ npm install -g elm
$ elm reactor
```

Then, naviagte to http://0.0.0.0:8000/Todo.elm.
